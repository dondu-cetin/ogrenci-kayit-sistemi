# Öğrenci Kayıt Sistemi

Bu proje, Python ve SQLite kullanılarak geliştirilmiş, CRUD işlemlerini destekleyen temel bir veri yönetim sistemidir.

## ✨ Özellikler
- **CRUD İşlemleri:** Ekleme, Okuma, Güncelleme, Silme.
- **Akıllı Arama:** `LIKE` operatörü ile ismin bir kısmından arama yapabilme.
- **Veri Güvenliği:** SQLite ile verilerin kalıcı olarak saklanması.
